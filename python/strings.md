# Strings 

## Invertendo uma string
Utilizando slices para isto
```python
text = "abba"
inverse = text[::-1]
```
## String interpolation (Only works with python 3.5+)
```python
variable = "good"
f"text is {variable}"
``` 
## String format
```python
variable = "good"
"text is {}".format(variable)
```
